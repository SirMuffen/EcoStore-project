 
# EcoStore

Projet réalisé dans le cadre du module Développement Web à l'EMSI
Casablanca, 2ème année IIR, année 2025/2026.
Auteurs : Ayoub ABIDA leader du projet et Youssef AL HAFRAWI

## Description

EcoStore est un site e-commerce dédié aux produits éco-responsables
au Maroc. Il a été développé entièrement en HTML5 et CSS3 sans aucun
framework, ce qui nous a permis de bien maîtriser les bases du
développement web.

Le site contient six pages : la boutique principale, une fiche produit
détaillée, un panier, un formulaire de paiement et confirmation, une page d'impact
écologique et une page de localisation avec Google Maps intégré.

## Techniquement

On a utilisé Flexbox et CSS Grid pour la mise en page, des variables
CSS pour les couleurs et les polices, et on a fait attention au design
responsive pour que le site s'affiche correctement sur mobile.

## Lancer le projet

Télécharger les fichiers et ouvrir index.html dans un navigateur.
Aucune installation requise.
